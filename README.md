# ひらがな迷路

ひらがなを集めながら迷路を攻略するブラウザゲームです。

**🎮 今すぐ遊ぶ → https://pndshch.github.io/hiragana-maze/**

---

## ゲームモード

### じゅんばんモード
迷路の中に散らばった **あ〜ん の46文字** を、あいうえお順に全部集めよう。  
次に取るべき文字は赤くひかるので、迷路を探索しながら目指してください。  
全部集めたら、ゴール（★）に向かえばクリア！

### ことば集めモード
お題の **ことば**（さくら・あおぞら・ひまわり など29種）が出題されます。  
その文字を順番に拾い、ゴールへ。デコイのひらがなに惑わされないように！

---

## 操作方法

| 環境 | 操作 |
|------|------|
| キーボード | 矢印キー（↑↓←→）または WASD |
| スマホ・タブレット | スワイプ または 画面下の十字ボタン |

### 読み上げ機能 🔊

ひらがなを踏むと音声で読み上げます。OS にインストールされた一番きれいな日本語ボイスを自動選択します。

- macOS / iOS: **Kyoko**（Premium 版があれば優先）
- Windows: **Microsoft Nanami / Ayumi / Haruka**
- Android Chrome: **Google 日本語**

正解を踏むと明るい高めの声、間違いだと少し低めの声、クリア時は「クリア！おめでとう！」と読み上げます。  
HUD の 🔊 ボタンで ON/OFF、HUD の次の文字をタップで再読み上げできます。

## ルール

- 赤くひかっている文字が **次に取るべき文字** です
- 正しい文字を踏むと収集、間違いは赤フラッシュ（ペナルティなし）
- 全文字を集めてからゴール（★）に入るとクリア
- タイムが記録されます（速いほど上手！）

---

## 技術スタック

- HTML / CSS / JavaScript（ビルドツールなし）
- Canvas 2D API でレンダリング
- 迷路は [再帰バックトラック法](https://en.wikipedia.org/wiki/Maze_generation_algorithm#Randomized_depth-first_search) でランダム生成
- GitHub Pages でホスティング
